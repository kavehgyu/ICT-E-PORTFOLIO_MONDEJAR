Lessons

Lesson 1: Introduction to CSS

CSS (Cascading Style Sheets): A stylesheet language used to describe the presentation of a document written in HTML or XML.

Importance: CSS enhances the design and layout of web pages, allowing for separation of content from presentation.

Lesson 2: Style Sheets

  • External Stylesheet: Linked using the <link> tag.
  • Embedded Styles: Defined within <style> tags.
  • Inline Styles: Applied directly in the style attribute.

Lesson 3: Display Property

Display Property: Allows you to control how an element is displayed

  • Block Level Element: Takes up as much space as possible, vertical layout
  • Inline Level Element: Display in a line, horizontal layout, cannot contain block elements
  • Importance: Some CSS properties react differently for each display type
  • Property values: inline, block, list-item, none, inherit
  • Div & Span Tags: Div for sections, span for inline emphasis

Lesson 4: Classes

  • User-defined selector applied to specific HTML tags
  • Reusable style for multiple elements
  • Element + Class selector allows precise styling
  • Grouping selectors: combine multiple selectors with a comma

Lesson 5: CSS Selectors

  • Element Selector: Styles HTML elements directly
  • ID Selector: Applies to one unique element (#id)
  • Universal Selector: Selects all elements (*)
  • Group Selector: Style multiple elements together
  • Attribute Selector: Selects elements with specific attribute values
